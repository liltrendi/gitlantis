## Getting Started

You need to install:

- [Node.js](https://nodejs.org/en/download/)
- [pnpm](https://pnpm.io/installation)
- [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Install

```bash
nvm use
pnpm i
```

### Develop

```bash
pnpm dev
```

### Build

#### Browser

```bash
pnpm run build:browser
```

#### Extension

```bash
pnpm run build:extension
```

#### Both

```bash
pnpm run build
```
